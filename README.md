# URL Shortener

A frontend-only URL Shortener built as part of an assessment.

## Features
- Enter long URL and generate short link  
- Copy short link  
- Simple and clean UI  
- Responsive design  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## How It Works
1. Enter a long URL  
2. Click "Shorten"  
3. Get shortened link  
4. Copy and share  

## Run Locally
Clone the repo and open index.html:
git clone <your-repo-link>
cd URL-Shortener
open index.html

## Purpose
Built as a frontend assessment project to test:
- UI design  
- JavaScript logic  
- API or mock handling  
- Code structure
